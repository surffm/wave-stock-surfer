## Instructions

- **Arrow keys**: Move surfer
- **Space**: Jump
- **Click a wave**: Select stock
- **Add new wave**: Click "Add New Wave"

## Local Dev

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. **Push all files to this repo**
2. In repo settings, go to Pages & select root of the `main` branch to publish.
3. Your game will be live at:  
   `https://surffm.github.io/wave-stock-surfer/`
